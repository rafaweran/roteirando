## Arquivo necessário:
- `logo.png` - Logo circular do Roteirando

A logo deve ser uma imagem PNG com fundo transparente ou com o design circular conforme descrito.

## Tamanho recomendado:
- Mínimo: 200x200px
- Ideal: 400x400px ou maior (será redimensionada automaticamente)
